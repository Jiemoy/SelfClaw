# SelfClaw

SelfClaw 是一个 OpenClaw 的图形界面客户端，基于 Tauri + React + TypeScript 构建。

## 项目简介

SelfClaw 提供了一个直观的图形用户界面，用于管理和使用 OpenClaw 功能。

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **桌面框架**: Tauri
- **样式**: Tailwind CSS
- **状态管理**: Zustand

## 开发语言

- TypeScript (73.4%)
- Rust (24.5%)
- JavaScript (1.5%)

## 项目结构

```
├── src/              # React 前端源代码
├── src-tauri/        # Tauri 后端源代码
├── public/           # 静态资源
├── dist/             # 构建输出
└── .github/          # GitHub 配置
```

## 安装依赖

```bash
npm install
```

## 开发模式

```bash
npm run dev
```

## 构建
